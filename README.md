## 命令

| 命令 | 介绍 | 权限 |
| ----- | ----- | ----- |
| /cuilian reload | 重载插件配置 | op |
| /cuilian | 获取命令的帮助 | all |
| /cuilian moveLevel | 打开转移淬炼等级至新的装备的箱子UI | all |
| /cuilian set [int: 等级] | 设置强化等级 | op |
| /cuilian stone [string: 淬炼石ID] [int: 数量] [string: 玩家] | 给予强化石 | op |
| /cuilian protect [int: 保护符等级] [int: 数量] [string: 玩家] | 给予保护符 | op |
